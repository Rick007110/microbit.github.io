Micro:bit Cheat Sheet

  • Forever:

    Dit is een commando dat de micro:bit vertelt om een blok code of functie continu uit te voeren, totdat het programma wordt gestopt of de stroom wordt uitgeschakeld.

  • leds:

    Dit commando wordt gebruikt om een patroon of ontwerp weer te geven met behulp van de LED's op de micro:bit. Je kunt dit commando gebruiken om een aangepaste animatie te maken of een specifieke boodschap weer te geven.

  • On shake:

    Dit commando wordt gebruikt om te detecteren wanneer de micro:bit wordt geschud. Je kunt dit commando gebruiken om een bepaalde actie of gebeurtenis te activeren wanneer het apparaat wordt geschud.

  • Show number:

    Dit commando wordt gebruikt om een getal weer te geven op de LED's van de micro:bit. Je kunt dit commando gebruiken om het resultaat van een berekening weer te geven of een waarde te tonen die in de tijd verandert.

  • Pick random:

    Dit commando wordt gebruikt om een willekeurig getal binnen een bepaald bereik te genereren. Je kunt dit commando gebruiken om een spel te maken of een willekeurige waarde te genereren voor een experiment.

  • Loops:

    Loops worden gebruikt om een blok code een bepaald aantal keren te herhalen of totdat aan een bepaalde voorwaarde is voldaan. Er zijn verschillende soorten loops in micro:bit, waaronder de "for" loop, de "while" loop en de "repeat" loop.

  • For index:

    Dit type loop wordt gebruikt om een blok code een specifiek aantal keren te herhalen. Je kunt deze loop gebruiken wanneer je weet hoe vaak je een blok code wilt herhalen.

  • While:

    Dit type loop wordt gebruikt om een blok code te herhalen zolang een bepaalde voorwaarde waar is. Je kunt deze loop gebruiken wanneer je een blok code wilt herhalen totdat aan een specifieke voorwaarde is voldaan.

  • Repeat … times:

    Dit commando wordt gebruikt om een blok code een bepaald aantal keren te herhalen. Je kunt dit commando gebruiken om een loop te maken die een bepaald aantal keren wordt herhaald.

  • For element value of:

    Dit type loop wordt gebruikt om een blok code voor elk element in een lijst of array te herhalen. Je kunt deze loop gebruiken wanneer u dezelfde bewerking op elk element in een lijst wilt uitvoeren.

  • Set…to…:

    Dit commando wordt gebruikt om een waarde toe te kennen aan een variabele. Je kunt dit commando gebruiken om de waarde van een variabele in te stellen.

  • Array:

    Een array is een lijst of verzameling van waarden die in de code kunnen worden gebruikt. Je kunt arrays gebruiken om meerdere waarden op te slaan en te bewerken.

  • If:

    Dit is een commando dat wordt gebruikt om een bepaalde actie uit te voeren als aan een bepaalde voorwaarde is voldaan.

  • Else:

    Dit is een commando dat wordt gebruikt als een alternatief voor de "if" -voorwaarde. Als de voorwaarde van de "if" niet waar is, kan de code in de "else" -sectie worden uitgevoerd.

  • For element value of:

    Dit type loop wordt gebruikt om een blok code voor elk element in een lijst of array te herhalen. Je kunt deze loop gebruiken wanneer u dezelfde bewerking op elk element in een lijst wilt uitvoeren.

  • For element value of

    Dit type loop wordt gebruikt om een blok code voor elk element in een lijst of array te herhalen. Je kunt deze loop gebruiken wanneer u dezelfde bewerking op elk element in een lijst wilt uitvoeren.

  • Set…to…

    Dit commando wordt gebruikt om een waarde toe te kennen aan een variabele. Je kunt dit commando gebruiken om de waarde van een variabele in te stellen.

  • Array

    Een array is een lijst of verzameling van waarden die in de code kunnen worden gebruikt. Je kunt arrays gebruiken om meerdere waarden op te slaan en te bewerken.

    Dit is een commando dat wordt gebruikt om een bepaalde actie uit te voeren als aan een bepaalde voorwaarde is voldaan.

  • Else

    Dit is een commando dat wordt gebruikt als een alternatief voor de "if" -voorwaarde. Als de voorwaarde van de "if" niet waar is, kan de code in de "else" -sectie worden uitgevoerd.

  • Else-if

    Dit is een commando dat wordt gebruikt wanneer u meerdere voorwaarden wilt controleren. Als de voorwaarde van de "if" niet waar is, kan de code in de "else-if" -sectie worden uitgevoerd.

  • Variabelen

    Een variabele is een container die waarden kan opslaan die in de code kunnen worden gebruikt. Je kunt variabelen gebruiken om waarden op te slaan en te bewerken die in de code worden gebruikt.

  • Datatypen

    Een datatype is een type gegevens dat kan worden opgeslagen in een variabele. Micro:bit ondersteunt verschillende datatypen, waaronder strings, floats, integers, doubles en booleans.

  • String

    Een string is een datatype dat wordt gebruikt om tekst op te slaan. Je kunt strings gebruiken om berichten weer te geven op de micro:bit LED's of om tekstgegevens op te slaan.

  • Float

    Een float is een datatype dat wordt gebruikt om decimale getallen op te slaan. Je kunt floats gebruiken voor nauwkeurige berekeningen die decimalen vereisen.

  • Integer

    Een integer is een datatype dat wordt gebruikt om gehele getallen op te slaan. Je kunt integers gebruiken voor berekeningen die alleen hele getallen vereisen.

  • Double

    Een double is een datatype dat wordt gebruikt om grote decimale getallen op te slaan. Je kunt doubles gebruiken voor nauwkeurige berekeningen die grote getallen vereisen.

  • Boolean

    Een boolean is een datatype dat waar of onwaar (true of false) kan zijn. Je kunt booleans gebruiken voor beslissingslogica in de code, zoals het bepalen van welke actie moet worden ondernomen op basis van een bepaalde voorwaarde.
